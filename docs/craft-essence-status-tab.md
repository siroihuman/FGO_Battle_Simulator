# 概念礼装状態の専用表示

概念礼装によって付与される継続状態は、サーヴァントカード上の通常バフ欄へ混在させず、クラススキル・クラススコアと同じ独立表示枠で管理します。

## 表示区分

サーヴァントカードのスキル欄直下に、存在する区分だけを表示します。

- クラススキル
- クラススコア
- 概念礼装

各区分は個別に開閉でき、開閉状態はサーヴァント単位で保持されます。

## 概念礼装状態の判定

状態アイコンの`source`と、`FGO_SIM_DATA.craftEssences`へ登録された概念礼装名を照合します。

一致した状態は通常バフ欄から取り外し、「概念礼装」表示枠へ移動します。

対象には強化だけでなく、概念礼装由来の継続デメリットも含まれます。例として「黒の聖杯」では次の2状態が同じ概念礼装枠へ表示されます。

- 宝具威力アップ
- 毎ターンHP減少

戦闘開始時NP増加のように、状態として残らず即時反映される効果はアイコン表示の対象外です。
