# エネミーのサーヴァント設定・クラス固定値

## 参照資料

Fate/Grand Order @wiki「考察/その他/エネミー」：

## ウェーブ設定画面

各エネミーの属性欄横へ、以下のチェックボックスを追加する。

- `サーヴァント`
- `チャージ設定`
- `クラス数値設定`

### サーヴァント

ONの場合、エネミーへ`サーヴァント`特性を追加する。

OFFの場合、チェックボックスによって追加された`サーヴァント`特性を削除する。

これにより、〔サーヴァント〕特攻・条件判定・クラス相性条件などからサーヴァントとして参照できる。

### チャージ設定

OFFの場合、選択クラスの固定チャージ数を使用し、チャージ入力欄を無効化する。

ONの場合、0～10の範囲で手動設定できる。

チャージ0は宝具・チャージ攻撃無効を表し、ゲージは増加せず、敵は宝具を使用しない。

### クラス数値設定

OFFの場合、選択クラスの以下の固定値を使用する。

- 行動回数
- 行動優先度
- クリティカル率
- DSR
- DTDR
- ATDR

ONの場合、各項目を手動設定できる。

DRはクラス固定値ではなくエネミー種別・レアリティ等によって決まるため、従来どおり常時手動設定とする。

## 敵行動順

敵全体で1ターン最大3回行動する。

1. 行動回数が1以上の生存エネミーが、画面左側から1回ずつ行動する。
2. 行動枠が余った場合、個別の行動回数上限に達していないエネミーから、行動優先度が最も高いものが追加行動する。
3. 優先度が同じ場合は、画面左側のエネミーを優先する。

例として、キャスターとアヴェンジャーが並んでいる場合は、次の順番になる。

```text
キャスター → アヴェンジャー → アヴェンジャー
```

行動回数を0に手動設定したエネミーは行動しない。

## 戦闘画面

敵カードへ現在チャージを表示する。

通常表示：

```text
CHARGE ●●○○ 2/4
```

チャージ0：

```text
CHARGE OFF
```
