# フォウ強化上限

## 上限

HPフォウとATKフォウの入力・保存・戦闘反映上限は、それぞれ3,000とする。

```text
通常フォウ最終値 = min(3,000, 入力値)
```

3,000を超える値が入力・設定された場合は、戦闘開始時に3,000へ丸める。

## グランドサーヴァント

グランドサーヴァントのHP／ATK+1,000はフォウ強化と同じ枠へ加算する。加算後の最終値にも3,000上限を適用する。

```text
グランド時フォウ最終値 = min(3,000, 通常フォウ値 + 1,000)
```

例：

| 通常フォウ | グランド加算 | 最終値 | 実効加算量 |
|---:|---:|---:|---:|
| 1,500 | +1,000 | 2,500 | 1,000 |
| 2,500 | +1,000 | 3,000 | 500 |
| 3,000 | +1,000 | 3,000 | 0 |

上限超過分は切り捨てる。HP／ATK+1,000を強化状態として付与するわけではないため、強化解除などの影響は受けない。

## UIと保存

`js/fou-cap-3000.js`が編成画面のHPフォウ・ATKフォウ入力欄を0～3,000へ拡張する。

既存の編成処理が2,000上限で保存する構造との互換性を保つため、3,000対応値は専用設定として保存し、戦闘生成時に編成順へ再適用する。ページ再表示時も専用設定から入力欄へ復元する。

## 実装順

`js/fou-cap-3000.js`は`js/grand-score-core.js`より後、`js/grand-score-chain.js`と`js/app.js`より前に読み込む。

これにより、グランド+1,000を解決した後のフォウ値へ3,000上限を適用し、編成画面が使用する`BattleEngine`にも上限対応済みクラスを渡す。
