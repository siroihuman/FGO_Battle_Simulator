# 宝具・コマンドカードの攻撃相性表示

宝具カードと通常コマンドカードには、現在選択中のエネミーに対するクラス攻撃相性を表示します。

## 表示規則

- クラス相性倍率が1より大きい：`WEAK`
- クラス相性倍率が1より小さい：`RESIST`
- クラス相性倍率が1：表示なし

判定には`FGO_SIM_ENGINE.classAffinity`を使用し、ダメージ計算で使用する基本クラス相性と同じ結果を参照します。

## 更新タイミング

- エネミーの選択変更
- スキル使用後の画面再描画
- コマンドカード選択・解除
- Wave移行
- 前衛交代

表示処理は`js/command-affinity-labels.js`、スタイルは`css/command-affinity-labels.css`で管理します。
