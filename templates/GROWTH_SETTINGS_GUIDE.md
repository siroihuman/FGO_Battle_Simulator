# 育成設定

各サーヴァントの `levelStats` に3段階の基礎HP/ATKを登録します。

```js
levelStats: {
  max: { hp: 12000, atk: 11000 },
  100: { hp: 13200, atk: 12100 },
  120: { hp: 15500, atk: 14200 }
}
```

- `max`：最終再臨時の通常レベル上限
- `100`：聖杯転臨1
- `120`：聖杯転臨2
- フォウ強化、概念礼装補正は戦闘開始時に自動加算されるため、ここには含めません。
- コマンドカード強化は編成設定値として処理され、宝具・Extra Attackには加算されません。
