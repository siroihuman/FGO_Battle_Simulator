# descriptionを改行する方法

最も簡単なのは、文字列をバッククォート ` で囲み、コード上でそのまま改行する方法です。

```js
description: `味方単体の攻撃力をアップ(3T)
＆クリティカル威力をアップ(3T)
＋スターを獲得する。`,
```

通常のシングルクォート `'...'` またはダブルクォート `"..."` を使う場合は `\n` を入れます。

```js
description: '味方単体の攻撃力をアップ(3T)\n＆クリティカル威力をアップ(3T)',
```

画面側は `title` 属性へ説明を渡しているため、どちらの方法でもマウスオーバー表示内で改行されます。
