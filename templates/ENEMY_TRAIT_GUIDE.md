# 敵特性の追加・利用方法

## 編成画面での入力

敵の「特性（記述式）」欄へ、次のように入力します。

```text
神性 / 王 / 人型 / 人類の脅威
```

区切りには `/`、`／`、`,`、`，`、改行を使用できます。前後の空白は自動で除去されます。

## 特攻を作る

```js
{ type: 'traitPowerUp', target: 'self', trait: '王', value: 50, duration: 3 }
```

敵の特性欄に `王` があれば反応します。大文字・小文字、全角・半角、括弧の有無も含めて完全一致です。

## 宝具の特攻

```js
special: { kind: 'trait', key: '人類の脅威', multiplier: 1.5 }
```

## 戦闘中に特性を付与

```js
{ type: 'addTrait', target: 'allEnemies', trait: 'チェック', value: 1, duration: 1, debuff: true }
```

現行版では、付与した特性はその戦闘中に特攻判定へ反映されます。

## 新しい特性を増やす場合

特性一覧への登録作業は不要です。敵入力・サーヴァントの `traits`・効果側の `trait` または `key` で同じ文字列を使用してください。
